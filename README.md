# 🧠 HTML & CSS Quiz Webpage

## 📌 Project Overview

This project demonstrates the creation of a **Quiz Webpage using HTML form elements**. The webpage collects student information and allows users to answer quiz questions related to **HTML and CSS concepts**.

The primary objective of this project is to practice **HTML forms, radio buttons, dropdown menus, textarea inputs, and structured webpage layout**.

This project highlights essential web development concepts including **form-based data collection, question structuring, input validation, and user interaction through HTML elements**.

---

## 🎯 Problem Statement

To design an interactive quiz webpage that collects student details and allows them to answer questions related to HTML and CSS using different form input types.

---

## 🛠️ Project Modules Covered

### 1️⃣ HTML Document Structure
- Created the webpage using `html`, `head`, and `body`.
- Added meta tags for character encoding and responsive layout.

### 2️⃣ Header Section
- Displayed the title **Quiz on HTML and CSS**.

### 3️⃣ Student Information Form
- Collected basic student details including:
  - Name
  - Email
  - Date of Birth

### 4️⃣ HTML Quiz Section
- Added quiz questions related to HTML.
- Used **radio buttons** for true/false answers.

### 5️⃣ CSS Quiz Section
- Included a question related to CSS properties.
- Used a **dropdown menu (select element)** for answer selection.

### 6️⃣ Feedback Section
- Added a **textarea** allowing users to ask questions or give feedback.

### 7️⃣ Form Submission
- Implemented a **submit button** to send quiz responses.

### 8️⃣ Footer Section
- Added contact details, copyright information, and website link.
